# Hola mundo en GitHub

Este repo incluye un ejemplo mínimo para que puedas verlo de dos formas:

1. **Directo en GitHub:** abre `index.html` y verás el código fuente.
2. **Como página renderizada:** activa **GitHub Pages** para que se muestre en el navegador.

## Cómo publicarlo con GitHub Pages

1. Sube este repo a GitHub.
2. Ve a **Settings → Pages**.
3. En **Build and deployment**, elige:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` y carpeta `/ (root)`
4. Guarda y espera 1–2 minutos.
5. GitHub te dará una URL tipo:

```text
https://TU-USUARIO.github.io/TU-REPO/
```

Al abrir esa URL, verás el **Hola mundo** renderizado.
